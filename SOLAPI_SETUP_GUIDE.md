## Solapi 발신번호 등록 가이드

### 문제 상황
- Solapi 콘솔에 "선택된 발신번호 없음" 표시
- SMS 발송 불가 상태

### 즉시 해결 방법

#### 1단계: Solapi 콘솔 접속
https://console.solapi.com/purplebook

#### 2단계: 발신번호 등록
1. 좌측 메뉴 → "발신번호" 클릭
2. "발신번호 추가" 버튼 클릭
3. 전화번호 입력: **01082411619**
4. 인증 방법 선택:
   - ARS 인증 (즉시 가능, 추천)
   - 서류 인증 (영업일 1-2일 소요)

#### 3단계: ARS 인증 절차
1. 전화번호 입력 후 "ARS 인증" 선택
2. 01082411619 번호로 전화 옴
3. 음성 안내에 따라 인증번호 입력
4. 인증 완료 → 즉시 사용 가능

#### 4단계: 등록 확인
- 발신번호 목록에 "01082411619" 표시
- 상태: "사용 가능" 또는 "인증 완료"

### 완료 후 테스트
1. 로컬 서버 재시작
